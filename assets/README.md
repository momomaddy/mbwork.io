# Fortune creative assets

Place the Fortune work images used by `fortune.html` in this folder with these names:

- `fortune-work-01.jpg`
- `fortune-work-02.jpg`
- `fortune-work-03.jpg`
- `fortune-work-04.jpg`
- `fortune-work-05-hero.jpg` (hero creative)

The last image is displayed both in the gallery and in the hero creative highlight block.
